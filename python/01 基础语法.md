## 0. 课前准备

### 0.1 装 Python

去 [python.org/downloads](https://www.python.org/downloads/) 下载 Python 3.11 或更新版本。Windows 安装时**务必勾选** "Add Python to PATH"，否则后面命令行会找不到 `python`。

装好之后打开终端（Windows 用 PowerShell 或 cmd，Mac 用 Terminal）：

```bash
python --version
```

看到类似 `Python 3.11.9` 就说明装好了。

### 0.2 装编辑器

推荐 **VSCode**（[code.visualstudio.com](https://code.visualstudio.com)），免费，安装后搜索并安装 "Python" 扩展。

### 0.3 两种运行 Python 的方式

**方式 1：交互式（适合试代码）**

在终端输入 `python`，回车，会出现 `>>>` 提示符：

```python
>>> print("Hello, World!")
Hello, World!
>>> 1 + 2
3
```

输入 `exit()` 退出。

**方式 2：运行脚本（适合写完整程序）**

新建一个文件 `hello.py`，内容：

```python
print("Hello, World!")
```

保存后，在终端里 `cd` 到文件所在目录，运行：

```bash
python hello.py
```

就会输出 `Hello, World!`。

## 1. 第一个程序：Hello, World!

任何编程语言的第一课都是输出"Hello, World!"。

```python
# 这是注释，计算机不会执行
# print() 是 Python 自带的"打印"函数，把内容显示在屏幕上
print("Hello, World!")
```

**讲解**：
- `#` 开头的行是**注释**，给你自己看的，Python 会忽略。
- `print(...)` 是**内置函数**，作用是把括号里的内容显示出来。
- `"Hello, World!"` 是一个**字符串**（一串文字），必须用单引号 `' '` 或双引号 `" "` 括起来。
- 末尾的换行是自动加的，每次 `print` 都会换到下一行。

**试一试**：
```python
print("你好，世界！")
print("我叫小明")
print(123)
print(3.14)
```

运行后会看到四行输出。

## 2. 变量：给数据起个名字

变量就是一个"盒子"，用来装数据，方便重复使用。

```python
# 把数据装到变量里
name = "小明"      # 字符串
age = 18           # 整数
height = 1.75      # 小数
is_student = True  # 布尔值（真/假）

# 用变量
print(name)        # 小明
print(age)         # 18
print(height)      # 1.75
print(is_student)  # True
```

**讲解**：
- `=` 的意思是"把右边的值放进左边的变量里"，**不是数学上的等于**。
- 变量名可以自己起，但有规则：
  - 只能用字母、数字、下划线；
  - 不能以数字开头；
  - 不能用 Python 的关键字（`if` `for` `class` 等）；
  - 推荐用有意义的英文名：`age` 比 `a` 好，`user_name` 比 `n1` 好。

**示例：变量可以变**

```python
score = 60
print(score)    # 60

score = 90     # 重新赋值
print(score)    # 90
```

---
**注意：`=` 是赋值运算符，不是等于运算符**
- `=` 是把右边的值赋给左边的变量，相当于"把右边的盒子放进左边的盒子里"。
- `==` 是比较运算符，判断两个值是否相等。
- a = 2 和 a == 2 是不同的，前者是赋值，后者是比较。

```python
a = 2         # 先在内存中创建整数对象 2，再让变量 a 作为“标签”，指向这个整数对象 2
b = a         # 让 b 也指向 a 当前绑定的对象，可通过 id(a) == id(b) 验证，两者指向同一内存地址的对象
a = 3         # 先创建新的整数对象 3，再让 a 这个“标签”从原来的对象 2 解绑，转而绑定到新对象 3
              # 但 b 仍然指向原来的对象 2（因为 b 的绑定关系没被修改）
print(b)      # 输出 2，b 始终指向对象 2，所以输出 2
```


## 3. 基本数据类型
https://www.runoob.com/python3/python3-data-type.html
Python 有 6 种基本类型，前 4 个最常用：

| 类型 | 名称 | 例子 | 说明 |
|------|------|------|------|
| `int` | 整数 | `18`, `-5`, `0` | 没有小数点的数 |
| `float` | 浮点数 | `3.14`, `-0.5` | 带小数点的数 |
| `str` | 字符串 | `"hello"`, `'你好'` | 文字，必须加引号 |
| `bool` | 布尔值 | `True`, `False` | 只有两个值：真/假 |
| `None` | 空值 | `None` | 表示"什么都没有" |
| `complex` | 复数 | `1+2j` | 很少用 |

**怎么知道一个变量是什么类型？** 用 `type()` 函数：

```python
print(type(18))        # <class 'int'>
print(type(3.14))      # <class 'float'>
print(type("hello"))   # <class 'str'>
print(type(True))      # <class 'bool'>
print(type(None))      # <class 'NoneType'>
```

**类型转换**：有时候需要把字符串变成数字，或者反过来。

```python
# 字符串 → 数字
a = int("18")          # 结果是整数 18
b = float("3.14")      # 结果是浮点数 3.14

# 数字 → 字符串
c = str(100)           # 结果是字符串 "100"

print(a + 2)           # 20  （数字相加）
print("我今年" + c + "岁")  # 我今年100岁 （字符串拼接）
```

**重要提示**：`+` 号在数字之间是"相加"，在字符串之间是"拼接"，不能混用：

```python
print("年龄：" + 18)   # 报错！TypeError
print("年龄：" + str(18))  # 正确：年龄：18
```

## 4. 运算符
https://www.runoob.com/python3/python3-basic-operators.html

### 4.1 算术运算符

```python
print(10 + 3)   # 13   加
print(10 - 3)   # 7    减
print(10 * 3)   # 30   乘
print(10 / 3)   # 3.333... 除（结果是小数）
print(10 // 3)  # 3    整除（只取整数部分）
print(10 % 3)   # 1    取余（除法的余数）
print(2 ** 10)  # 1024 幂（2 的 10 次方）
```

**小例子**：判断一个数是奇数还是偶数，看它除以 2 的余数：

```python
n = 17
if n % 2 == 0:
    print(f"{n} 是偶数")
else:
    print(f"{n} 是奇数")
```

### 4.2 比较运算符（结果是 True 或 False）

```python
print(5 == 5)   # True   等于
print(5 != 3)   # True   不等于
print(5 > 3)    # True   大于
print(5 < 3)    # False  小于
print(5 >= 5)   # True   大于等于
print(5 <= 4)   # False  小于等于
```

### 4.3 逻辑运算符
| 运算符 | 名称 | 例子 | 结果 | 说明 |
|--------|------|------|------|------|
| `and` | 与 | `True and False` | `False` | 两者都真才为真 |
| `or` | 或 | `False or True` | `True` | 有一个真就为真 |
| `not` | 非 | `not True` | `False` | 取反，真假互换 |
### 4.4 位运算符

| 运算符 | 名称 | 例子 | 结果 | 说明 |
|--------|------|------|------|------|
| `&` | 按位与 | `5 & 3` | `1` | 两位都为1才为1 |
| `|` | 按位或 | `5 | 3` | `7` | 有一位为1就为1 |
| `^` | 按位异或 | `5 ^ 3` | `6` | 两位不同才为1 |
| `~` | 按位取反 | `~5` | `-6` | 0变1，1变0 |
| `<<` | 左移 | `2 << 3` | `16` | 所有位向左移3位 |
| `>>` | 右移 | `8 >> 2` | `2` | 所有位向右移2位 |


```python
# and：两者都真才为真
print(True and True)    # True
print(True and False)   # False

# or：有一个真就为真
print(False or True)    # True
print(False or False)   # False

# not：取反
print(not True)         # False
print(not False)        # True
```

```python
# 练习：打印 1-5 和 1-10 的 XOR 表
# 在运算时是将数字转换成二进制再进行比较运算的
a = list(range(1, 6))
b = list(range(1, 11))
for i in a:
    for j in b:
        c = i ^ j
        print(f"{i} XOR {j} = {c}")
```

**实际例子**：

```python
age = 20
score = 85
# 录取条件：年龄>=18 且 分数>=60
if age >= 18 and score >= 60:
    print("录取")
else:
    print("不录取")
```

## 5. 字符串
https://www.runoob.com/python3/python3-string.html

字符串就是一串文字，是最常用的类型之一。

### 5.1 三种写法

```python
s1 = 'hello'         # 单引号
s2 = "hello"         # 双引号
s3 = """第一行
第二行
第三行"""              # 三引号，可以跨行

print(s1)    # hello
print(s2)    # hello
print(s3)    # 第一行\n第二行\n第三行
```

单引号和双引号基本一样，区别是字符串里有什么引号，外面就用另一种：

```python
print("他说：'你好'")     # 他说：'你好'
print('他说："你好"')     # 他说："你好"
```

### 5.2 转义字符

如果字符串里必须有同种引号，用 `\` 转义：

```python
print("他说：\"你好\"")   # 他说："你好"
print('It\'s a book')     # It's a book
print("第一行\n第二行")   # \n 表示换行
print("姓名\t年龄")        # \t 表示 Tab 缩进
```

### 5.3 字符串拼接

```python
# 方法 1：用 + 号
name = "小明"
greeting = "你好，" + name + "！"
print(greeting)   # 你好，小明！

# 方法 2：用 f-string（强烈推荐，更直观）
age = 18
print(f"我叫{name}，今年{age}岁")   # 我叫小明，今年18岁

# 可以在 {} 里做运算
print(f"明年我{age + 1}岁")         # 明年我19岁

# 可以控制小数位数
pi = 3.1415926
print(f"圆周率约等于 {pi:.2f}")     # 圆周率约等于 3.14
```

### 5.4 常用字符串方法


方法就是"挂在变量后面的小函数"，用 `.` 调用。

```python
s = "Hello, World!"

print(s.lower())         # hello, world!       全部小写
print(s.upper())         # HELLO, WORLD!       全部大写
print(s.strip())         # 去掉首尾空格
print(s.split(","))      # ['Hello', ' World!']  按逗号切分成列表
print(s.replace("World", "Python"))  # Hello, Python!  替换
print(s.startswith("Hello"))  # True  是否以...开头
print(s.endswith("!"))    # True  是否以...结尾
print(len(s))             # 13   字符串长度（注意 len 是函数不是方法）
```

**小例子**：判断手机号格式

```python
phone = "13812345678"
if len(phone) == 11 and phone.isdigit():
    print("手机号格式正确")
else:
    print("手机号格式错误")
```

age = input("请输入年龄：")

## 7. 控制流：让程序做选择和重复

### 7.1 if 条件判断

**基本结构**：

```python
if 条件:
    # 条件为真时执行
elif 另一个条件:
    # 上面条件不满足、这个条件满足时执行
else:
    # 上面都不满足时执行
```

**注意**：缩进（4 个空格）必须严格统一，Python 用缩进表示代码块。

**示例 1：判断成绩等级**

```python
score = float(input("请输入成绩："))

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

**示例 2：判断闰年**

```python
year = int(input("请输入年份："))
# 闰年：能被 4 整除但不能被 100 整除，或者能被 400 整除
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year} 是闰年")
else:
    print(f"{year} 不是闰年")
```

### 7.2 for 循环：把一件事做 N 次

**示例 1：用 range 控制次数**

```python
# 打印 0 到 4
for i in range(5):
    print(i)
# 输出 0 1 2 3 4，每个一行

# 打印 1 到 10
for i in range(1, 11):
    print(i)

# 打印 1, 3, 5, 7, 9
for i in range(1, 10, 2):
    print(i)
```

`range(start, stop, step)`：
- `start`：起始（包含），默认 0。
- `stop`：结束（不包含）。
- `step`：步长，默认 1。

**示例 2：遍历字符串**

```python
for ch in "Hello":
    print(ch)
# 依次打印 H e l l o
```

**示例 3：累加 1 到 100**

```python
total = 0
for i in range(1, 101):
    total = total + i
print(f"1+2+...+100 = {total}")   # 5050
```

**示例 4：九九乘法表**

```python
for i in range(1, 10):           # 行
    for j in range(1, i + 1):    # 列
        print(f"{j}*{i}={i*j}", end="\t")
    print()  # 换行
```

输出：
```
1*1=1
1*2=2   2*2=4
1*3=3   2*3=6   3*3=9
...
```

### 7.3 while 循环：满足条件就重复

```python
n = 1
while n <= 5:
    print(n)
    n = n + 1
# 输出 1 2 3 4 5
```

**经典例子：猜数字游戏**

```python
import random
answer = random.randint(1, 100)   # 随机生成 1~100 的整数

while True:
    guess = int(input("猜一个 1~100 的数字："))
    if guess > answer:
        print("大了")
    elif guess < answer:
        print("小了")
    else:
        print("猜对了！")
        break   # 跳出循环
```

### 7.4 break / continue / pass

```python
# break：跳出整个循环
for i in range(10):
    if i == 5:
        break
    print(i)
# 输出 0 1 2 3 4

# continue：跳过本轮，进入下一轮
for i in range(10):
    if i == 5:
        continue
    print(i)
# 输出 0 1 2 3 4 6 7 8 9

# pass：占位，什么都不做
for i in range(10):
    pass   # 还没想好写什么，先放这里
```

## 8. 函数：把代码打包

函数就是把一段代码起个名字，以后可以反复调用。

### 8.1 定义和调用

```python
# 定义
def say_hello():
    print("Hello!")
    print("欢迎学习 Python")

# 调用
say_hello()    # Hello!\n欢迎学习 Python
say_hello()    # 再调用一次
```

### 8.2 参数和返回值

```python
# 接收参数
def greet(name):
    print(f"你好，{name}！")

greet("小明")     # 你好，小明！
greet("小红")     # 你好，小红！

# 返回值
def add(a, b):
    return a + b

result = add(3, 5)
print(result)     # 8
```

### 8.3 默认参数

```python
def greet(name, prefix="你好"):
    print(f"{prefix}，{name}！")

greet("小明")              # 你好，小明！
greet("小红", "Hi")        # Hi，小红！
greet("小刚", prefix="Hello")  # Hello，小刚！  关键字参数
```

### 8.4 返回多个值

```python
def min_max(numbers):
    return min(numbers), max(numbers)

# 用元组解包接收
lo, hi = min_max([3, 1, 4, 1, 5, 9, 2, 6])
print(f"最小值：{lo}，最大值：{hi}")
```

### 8.5 实际例子：素数判断函数

```python
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):# 只需要判断到 sqrt(n) 即可，因为只是判断是否有因子，因子一定是成对出现的，不用全部循环
        if n % i == 0:
            return False
    return True

# 测试
for n in range(1, 21):
    if is_prime(n):
        print(n, end=" ")
# 输出 2 3 5 7 11 13 17 19
```

## 9. 常用内置函数

这些是 Python 自带的、随时可以用的函数，必须熟记：

| 函数 | 作用 | 例子 |
|------|------|------|
| `len(x)` | 长度 | `len("hello")` → `5` |
| `max(x)` | 最大值 | `max([3, 1, 4])` → `4` |
| `min(x)` | 最小值 | `min([3, 1, 4])` → `1` |
| `sum(x)` | 求和 | `sum([1, 2, 3])` → `6` |
| `sorted(x)` | 排序 | `sorted([3, 1, 4])` → `[1, 3, 4]` |
| `abs(x)` | 绝对值 | `abs(-5)` → `5` |
| `round(x, n)` | 四舍五入 | `round(3.14159, 2)` → `3.14` |
| `range(a, b, s)` | 整数序列 | `range(0, 10, 2)` → `0,2,4,6,8` |
| `enumerate(x)` | 编号遍历 | 见下方例子 |
| `zip(a, b)` | 并行遍历 | 见下方例子 |
| `map(f, x)` | 对每个元素应用 f | `list(map(int, "123"))` → `[1,2,3]` |
| `filter(f, x)` | 过滤 | `list(filter(lambda x: x>0, [-1,0,1]))` → `[1]` |

**enumerate 示例**：同时拿到元素和下标。

```python
fruits = ["苹果", "香蕉", "橙子"]
for i, fruit in enumerate(fruits):
    print(f"第{i+1}个水果是{fruit}")
# 第1个水果是苹果
# 第2个水果是香蕉
# 第3个水果是橙子
```

**zip 示例**：两个列表同时遍历。

```python
names = ["张三", "李四", "王五"]
scores = [85, 92, 78]

for name, score in zip(names, scores):
    print(f"{name} 考了 {score} 分")
```

## 10. 完整实战：学生成绩管理

把前面学的内容串起来，写一个完整的小程序：

```python
def input_students():
    """录入学生信息，返回学生列表"""
    students = []
    n = int(input("请输入学生人数："))
    for i in range(n):
        name = input(f"请输入第{i+1}个学生姓名：")
        score = float(input(f"请输入{name}的成绩："))
        students.append({"name": name, "score": score})
    return students


def show_summary(students):
    """显示统计信息"""
    if not students:
        print("没有学生数据")
        return

    scores = [s["score"] for s in students]
    avg = sum(scores) / len(scores)
    best = max(students, key=lambda s: s["score"])
    worst = min(students, key=lambda s: s["score"])

    print("\n===== 统计结果 =====")
    print(f"学生人数：{len(students)}")
    print(f"平均分：{avg:.2f}")
    print(f"最高分：{best['name']} ({best['score']})")
    print(f"最低分：{worst['name']} ({worst['score']})")


def show_rank(students):
    """按成绩从高到低排名"""
    print("\n===== 成绩排名 =====")
    ranked = sorted(students, key=lambda s: s["score"], reverse=True)
    for i, s in enumerate(ranked, start=1):
        print(f"第{i}名：{s['name']} {s['score']} 分")


# 主程序
students = input_students()
show_summary(students)
show_rank(students)
```

## 11. 易错点速查表

| 易错点 | 错误示例 | 正确写法 |
|--------|----------|----------|
| `input()` 直接当数字 | `n = input(); n + 1` | `n = int(input()); n + 1` |
| 缩进不一致 | 上一行 4 空格，下一行 1 Tab | 全用 4 空格（VSCode 设置 Tab Size=4） |
| 混淆 `=` 和 `==` | `if a = 5:` （赋值） | `if a == 5:` （比较） |
| 浮点数比较 | `0.1 + 0.2 == 0.3` → False | `abs(a-b) < 1e-9` |
| 字符串和数字相加 | `"age: " + 18` | `"age: " + str(18)` 或 `f"age: {18}"` |
| 修改字符串 | `s = "hello"; s[0] = "H"` | `s = "H" + s[1:]` |
| 列表复制陷阱 | `b = a; b.append(9)` 后 a 也变 | `b = a.copy()` 或 `b = a[:]` |
| 忘记冒号 | `if x > 0` | `if x > 0:` |
| range 结束值 | `range(1, 10)` 以为到 10 | 实际到 9（不包含结束值） |

### 项目 1：温度转换器

输入华氏温度，输出摄氏温度。公式：`C = (F - 32) × 5/9`

```python
f = float(input("请输入华氏温度："))
c = (f - 32) * 5 / 9
print(f"华氏 {f}°F = 摄氏 {c:.2f}°C")
```

### 项目 2：判断素数

输入一个正整数，判断是否为素数（除了 1 和自己之外没有其他因数）。

```python
n = int(input("请输入一个正整数："))
is_prime = True
if n < 2:
    is_prime = False
else:
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            is_prime = False
            break

if is_prime:
    print(f"{n} 是素数")
else:
    print(f"{n} 不是素数")
```

### 项目 3：九九乘法表

```python
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{j}*{i}={i*j}", end="\t")
    print()
```

### 项目 4：猜数字游戏（带次数限制）

```python
import random
answer = random.randint(1, 100)
max_times = 7

for i in range(1, max_times + 1):
    guess = int(input(f"第{i}次猜（剩{max_times - i}次）："))
    if guess > answer:
        print("大了")
    elif guess < answer:
        print("小了")
    else:
        print(f"恭喜，用了{i}次猜对！")
        break
else:
    print(f"很遗憾，正确答案是 {answer}")
```

### 项目 5：回文判断

回文：正着读和反着读一样，比如 "abcba"、"12321"。

```python
s = input("请输入字符串：")
# 方法 1：字符串反转后比较
if s == s[::-1]:
    print(f"「{s}」是回文")
else:
    print(f"「{s}」不是回文")
```

> 注：`s[::-1]` 是 Python 的"切片反转"语法，第 ② 阶段会详细讲。
