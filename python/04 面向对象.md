## 3. 知识点详细讲解

### 3.1 什么是面向对象

**类 = 图纸，对象 = 按图纸造出来的实物。**

想象你要造一扇门。你先画一张设计图（类），上面标注了门的尺寸、材质、颜色（属性），以及"开门""关门"（方法）。然后你按图纸造出 10 扇门（10 个对象），每扇门都有自己的颜色，但结构都一样。

**面向过程 vs 面向对象**：

- 面向过程：一步一步写"做什么"——洗菜 → 切菜 → 炒菜 → 装盘
- 面向对象：先造一个「厨师」，他自带洗菜、切菜、炒菜、装盘的本事。要炒菜？找个厨师来就行。

```python
# ===== 面向过程的写法 =====
def 洗菜(菜名):
    print(f"正在洗{菜名}")

def 切菜(菜名):
    print(f"正在切{菜名}")

def 炒菜(菜名):
    print(f"正在炒{菜名}")

def 装盘(菜名):
    print(f"正在装盘: {菜名}")

# 调用时一步步来
洗菜("白菜")
切菜("白菜")
炒菜("白菜")
装盘("白菜")

# ===== 面向对象的写法 =====
class 厨师:
    def __init__(self, 名字):
        self.名字 = 名字       # 厨师的属性

    def 做菜(self, 菜名):
        """一口气做完一道菜"""
        print(f"厨师{self.名字}开始做菜啦!")
        print(f"  1. 正在洗{菜名}")
        print(f"  2. 正在切{菜名}")
        print(f"  3. 正在炒{菜名}")
        print(f"  4. 装盘: {菜名}")

# 找一个厨师来做
王师傅 = 厨师("王师傅")
王师傅.做菜("白菜")
```

用面向对象的写法，以后要加新厨师、新菜，改起来方便得多。

### 3.2 定义类和创建对象

#### 基本语法

```python
class Dog:
    """狗类：一张'图纸'，描述狗是什么、能做什么"""

    # 类属性：所有狗都共享（写在类里，不在 __init__ 里面）
    species = "犬科动物"

    def __init__(self, name, age):
        """初始化方法：每次 '造狗' 时自动调用
        self 指代当前正在创建的那只狗
        name、age 是创建时传进来的参数"""
        self.name = name    # 实例属性：每只狗都不一样
        self.age = age

    def bark(self):
        """实例方法：狗的行为"""
        print(f"{self.name}('汪!汪!' - {self.age}岁)")

# ===== 创建对象（按图纸造狗）=====
# 类名用大驼峰命名法：每个单词首字母大写
wang_cai = Dog("旺财", 3)    # 自动调用 __init__(wang_cai, "旺财", 3)
xiao_hei = Dog("小黑", 1)

wang_cai.bark()    # 输出：旺财('汪!汪!' - 3岁)
xiao_hei.bark()    # 输出：小黑('汪!汪!' - 1岁)

# 访问属性
print(wang_cai.name)         # 旺财
print(wang_cai.age)          # 3
print(wang_cai.species)      # 犬科动物（类属性，所有实例共享）
print(Dog.species)           # 犬科动物（也可以通过类名直接访问）
```

#### self 到底是个啥？

`self` 是 Python 自动传进去的第一个参数，**代表"当前这个对象自己"**。你写 `wang_cai.bark()` 时，Python 自动变成 `Dog.bark(wang_cai)`。

```python
class Demo:
    def __init__(self, x):
        self.x = x
        print(f"__init__ 收到的 self 是: {self}")

    def show(self):
        print(f"show 收到的 self 是: {self}, x = {self.x}")

obj = Demo(10)      # __init__ 收到的 self 是: <__main__.Demo object at 0x...>
obj.show()          # show 收到的 self 是: <__main__.Demo object at 0x...>
                    # 两次 self 是同一个对象!
```

#### 实例属性 vs 类属性

| 对比项 | 实例属性 | 类属性 |
|--------|----------|--------|
| 定义位置 | `__init__` 里，用 `self.xxx` | 类体里直接写 |
| 归属 | 每个对象独立拥有 | 所有对象共享 |
| 修改影响 | 只影响当前对象 | 通过实例修改只影响实例，通过类修改影响所有实例 |
| 访问方式 | `对象.属性` | `对象.属性` 或 `类名.属性` |

```python
class Cat:
    species = "猫科动物"   # 类属性

    def __init__(self, name):
        self.name = name   # 实例属性

cat1 = Cat("小白")
cat2 = Cat("小花")

# 起初都共享类属性
print(cat1.species)   # 猫科动物
print(cat2.species)   # 猫科动物

# 通过对象修改——只影响 cat1（Python 给 cat1 新建了一个实例属性）
cat1.species = "神奇动物"
print(cat1.species)   # 神奇动物（现在是 cat1 自己的实例属性了）
print(cat2.species)   # 猫科动物（cat2 还是指向类属性）

# 通过类修改——影响所有还没被"覆盖"的实例
Cat.species = "喵星人"
print(cat1.species)   # 神奇动物（cat1 已有同名实例属性，不受影响）
print(cat2.species)   # 喵星人（cat2 没有同名实例属性，跟着类变）
```

### 3.3 方法

Python 类里有三种方法，每种方法的第一个参数和用途都不同。

| 方法类型 | 装饰器 | 第一个参数 | 调用方式 | 什么时候用 |
|----------|--------|------------|----------|-----------|
| 实例方法 | 无 | `self`（实例本身） | `对象.方法()` | 需要访问实例属性时（最常见） |
| 类方法 | `@classmethod` | `cls`（类本身） | `类名.方法()` 或 `对象.方法()` | 需要操作类属性，或作为"工厂方法"创建对象时 |
| 静态方法 | `@staticmethod` | 无特殊参数 | `类名.方法()` 或 `对象.方法()` | 与类相关但不需要访问实例/类属性的工具函数 |

```python
class Student:
    school = "星辰中学"      # 类属性
    student_count = 0       # 统计学生总数

    def __init__(self, name, grade):
        self.name = name    # 实例属性
        self.grade = grade
        Student.student_count += 1    # 每创建一个学生，总数 +1

    # ===== 实例方法：用到 self，操作具体某个学生 =====
    def introduce(self):
        """每个学生介绍自己"""
        print(f"大家好，我是{self.name}，来自{self.school}{self.grade}年级")

    # ===== 类方法：用到 cls，操作类级别的数据 =====
    @classmethod
    def get_count(cls):
        """查看全校有多少学生"""
        print(f"当前共有 {cls.student_count} 名学生")

    @classmethod
    def from_string(cls, info_str):
        """工厂方法：从字符串 "张三-3" 创建学生对象（不用手动拆字符串）"""
        name, grade = info_str.split("-")
        return cls(name, int(grade))    # cls(...) 等价于 Student(...)

    # ===== 静态方法：一个跟学生相关的工具函数，但不用 self 也不用 cls =====
    @staticmethod
    def is_valid_grade(grade):
        """检查年级是否合法（1~9）"""
        return 1 <= grade <= 9

# ===== 使用 =====
s1 = Student("张三", 3)
s2 = Student("李四", 5)
s3 = Student.from_string("王五-7")    # 用类方法快速创建

s1.introduce()          # 大家好，我是张三，来自星辰中学3年级
s3.introduce()          # 大家好，我是王五，来自星辰中学7年级
Student.get_count()     # 当前共有 3 名学生
print(Student.is_valid_grade(10))   # False（10年级不存在）
print(Student.is_valid_grade(5))    # True
```

### 3.4 封装

**封装 = 把数据藏起来，只暴露必要的接口。** 防止外面乱改数据导致出错。

#### 约定私有（单下划线 `_name`）

这只是程序员之间的约定——"别看这个，这是内部用的"，Python 不会真的阻止你。

```python
class Person:
    def __init__(self, name):
        self._name = name   # 约定的私有属性：建议不要直接访问

    def get_name(self):
        return self._name   # 通过公开方法获取值

p = Person("张三")
print(p.get_name())   # 正确方式：通过方法
print(p._name)        # 也能访问，但不建议（IDE 会警告你）
```

#### 强制私有（双下划线 `__name`）

Python 做了名称改写（name mangling），把 `__name` 改成 `_类名__name`，外面直接访问不到。

```python
class BankAccount:
    def __init__(self, owner, balance):
        self.owner = owner
        self.__balance = balance   # 余额是私有属性，不能直接改

    def deposit(self, amount):
        """存款"""
        if amount <= 0:
            print("存款金额必须大于 0")
            return
        self.__balance += amount
        print(f"存款成功! 存入 {amount} 元，当前余额 {self.__balance} 元")

    def withdraw(self, amount):
        """取款：余额不足时拒绝"""
        if amount <= 0:
            print("取款金额必须大于 0")
            return
        if amount > self.__balance:
            print(f"余额不足! 当前余额 {self.__balance} 元")
            return
        self.__balance -= amount
        print(f"取款成功! 取出 {amount} 元，当前余额 {self.__balance} 元")

    def get_balance(self):
        """查询余额（只读）"""
        return self.__balance

# ===== 使用 =====
acc = BankAccount("小明", 1000)
acc.deposit(500)         # 存款成功! 存入 500 元，当前余额 1500 元
acc.withdraw(200)        # 取款成功! 取出 200 元，当前余额 1300 元
acc.withdraw(2000)       # 余额不足! 当前余额 1300 元
print(acc.get_balance()) # 1300（正确方式）

# acc.__balance = -999   # 这不会改到真正余额，而是给 acc 加了个新属性
# print(acc.__balance)   # AttributeError: 'BankAccount' object has no attribute '__balance'
# 真正改余额需要用方法，外面没法直接乱改！
```

#### @property 装饰器：让方法像属性一样用

你不想让人乱改余额，但又希望 `acc.balance` 这种写法能直接拿到值（不用写 `acc.get_balance()`）。

```python
class BankAccount:
    def __init__(self, owner, balance):
        self.owner = owner
        self.__balance = balance   # 内部还是用 __balance

    @property
    def balance(self):
        """余额：只读属性（有 @property 但没有 @xxx.setter）"""
        return self.__balance

    @balance.setter
    def balance(self, value):
        """余额：设置时做检查"""
        if value < 0:
            raise ValueError("余额不能为负数!")
        self.__balance = value

# ===== 使用 =====
acc = BankAccount("小明", 1000)
print(acc.balance)     # 1000（像访问属性一样简单）
acc.balance = 2000     # 调用的其实是 setter
print(acc.balance)     # 2000
# acc.balance = -100   # ValueError: 余额不能为负数!
```

### 3.5 继承

**继承 = 子类自动获得父类的所有属性和方法**，然后在子类上做"加法"（新增）或"覆盖"（重写）。避免重复写相同代码。

#### 单继承

```python
# ===== 父类（基类）=====
class Animal:
    def __init__(self, name):
        self.name = name

    def eat(self):
        print(f"{self.name} 正在吃东西")

    def sleep(self):
        print(f"{self.name} 正在睡觉")

# ===== 子类：继承了 Animal 的所有代码 =====
class Dog(Animal):
    """Dog 继承 Animal，自动拥有 eat() 和 sleep()"""

    def bark(self):
        print(f"{self.name} 汪汪叫")    # 子类新增方法

    # 重写睡觉方法——狗睡觉可能是蜷缩的
    def sleep(self):
        print(f"{self.name} 蜷缩成一团睡着了")

class Cat(Animal):
    def meow(self):
        print(f"{self.name} 喵喵叫")

    def sleep(self):
        print(f"{self.name} 钻进纸箱睡着了")

# ===== 使用 =====
dog = Dog("旺财")
cat = Cat("小花")

dog.eat()      # 来自 Animal：旺财 正在吃东西
dog.bark()     # Dog 自己的：旺财 汪汪叫
dog.sleep()    # 重写后的：旺财 蜷缩成一团睡着了

cat.eat()      # 来自 Animal：小花 正在吃东西
cat.meow()     # Cat 自己的：小花 喵喵叫
cat.sleep()    # 重写后的：小花 钻进纸箱睡着了
```

#### super() 调用父类方法

当子类 `__init__` 需要在父类基础上"加点东西"时，用 `super()`：

```python
class Animal:
    def __init__(self, name):
        self.name = name

class Dog(Animal):
    def __init__(self, name, breed):
        # super() 指向父类 Animal，调用它的 __init__ 设置 name
        super().__init__(name)
        self.breed = breed       # 子类特有的属性

    def info(self):
        print(f"名字: {self.name}  品种: {self.breed}")

dog = Dog("大黄", "金毛")
dog.info()    # 名字: 大黄  品种: 金毛
```

#### 多继承 + MRO

Python 支持一个类同时继承多个父类。当多个父类有同名方法时，按 **MRO（方法解析顺序）** 决定用哪一个。

```python
class A:
    def say(self):
        print("来自 A")

class B:
    def say(self):
        print("来自 B")

class C(A, B):
    """先继承 A，再继承 B"""
    pass

c = C()
c.say()              # 来自 A（A 写在前面，优先匹配）
print(C.mro())       # [C, A, B, object]：按这个顺序找方法
```

#### isinstance() 和 issubclass()

```python
class Animal: pass
class Dog(Animal): pass

wang_cai = Dog()
print(isinstance(wang_cai, Dog))       # True
print(isinstance(wang_cai, Animal))    # True（子类实例也是父类类型）
print(issubclass(Dog, Animal))        # True（Dog 是 Animal 的子类）
print(issubclass(Animal, Dog))        # False
```

### 3.6 多态

**多态 = 不同对象对同一个方法名有不同的行为。** Python 不需要显式声明，只要对象有那个方法就能调用——这叫「鸭子类型」。

> "如果它走起来像鸭子，叫起来像鸭子，那它就是鸭子。"

```python
class Duck:
    def speak(self):
        print("嘎嘎嘎!")

class Dog:
    def speak(self):
        print("汪汪汪!")

class Cat:
    def speak(self):
        print("喵喵喵!")

# ===== 多态体现：同一个函数，传什么对象就有什么行为 =====
def animal_speak(animal):
    """只管调用 speak()，不关心 animal 到底是什么类型"""
    animal.speak()

# 任何有 speak() 方法的对象都能传
animal_speak(Duck())   # 嘎嘎嘎!
animal_speak(Dog())    # 汪汪汪!
animal_speak(Cat())    # 喵喵喵!

# ===== 更有用的例子：无需检查类型 =====
class WeChatPay:
    def pay(self, amount):
        print(f"微信支付: {amount} 元")

class AliPay:
    def pay(self, amount):
        print(f"支付宝支付: {amount} 元")

class Cash:
    def pay(self, amount):
        print(f"现金支付: {amount} 元")

def checkout(payment_method, amount):
    """结账：不管什么支付方式，有 pay 方法就行"""
    payment_method.pay(amount)

# 以后加新的支付方式，checkout 函数一行都不用改！
checkout(WeChatPay(), 100)   # 微信支付: 100 元
checkout(AliPay(), 200)      # 支付宝支付: 200 元
checkout(Cash(), 50)         # 现金支付: 50 元
```

### 3.7 魔术方法（Dunder Methods）

魔术方法以双下划线开头和结尾（`__xxx__`），让自定义对象能用上 Python 内置功能。

#### __str__ vs __repr__

```python
class Book:
    def __init__(self, title, author, price):
        self.title = title
        self.author = author
        self.price = price

    def __str__(self):
        """给用户看的友好输出，print() 和 str() 会调用"""
        return f"《{self.title}》作者: {self.author}  价格: {self.price}元"

    def __repr__(self):
        """给开发者看的调试输出，直接输入对象名时显示"""
        return f"Book('{self.title}', '{self.author}', {self.price})"

b = Book("Python入门", "张三", 59.9)
print(b)        # 《Python入门》作者: 张三  价格: 59.9元
print(repr(b))  # Book('Python入门', '张三', 59.9)
```

#### 比较运算符

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def __eq__(self, other):
        """== 判断成绩相等"""
        return self.score == other.score

    def __lt__(self, other):
        """< 判断成绩高低"""
        return self.score < other.score

    def __str__(self):
        return f"{self.name}({self.score}分)"

s1 = Student("张三", 85)
s2 = Student("李四", 90)
s3 = Student("王五", 85)

print(s1 == s3)    # True（成绩相同）
print(s1 == s2)    # False
print(s1 < s2)     # True（85 < 90）
```

#### 运算符重载

```python
class Vector:
    """二维向量，支持 + 和乘法运算"""

    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        """向量加法：v1 + v2"""
        return Vector(self.x + other.x, self.y + other.y)

    def __mul__(self, scalar):
        """向量数乘：v * 3"""
        return Vector(self.x * scalar, self.y * scalar)

    def __str__(self):
        return f"Vector({self.x}, {self.y})"

v1 = Vector(1, 2)
v2 = Vector(3, 4)
print(v1 + v2)       # Vector(4, 6)
print(v1 * 3)        # Vector(3, 6)
```

#### 索引访问 __getitem__ / __setitem__

```python
class MyList:
    """一个简单的容器，支持索引访问"""

    def __init__(self):
        self._data = []

    def __setitem__(self, index, value):
        """支持 obj[0] = xxx"""
        self._data[index] = value

    def __getitem__(self, index):
        """支持 x = obj[0]"""
        return self._data[index]

    def append(self, item):
        self._data.append(item)

ml = MyList()
ml.append("a")
ml.append("b")
ml.append("c")
print(ml[1])       # b
ml[2] = "hello"
```

### 3.8 抽象类 ABC

**抽象类 = 只定义接口，不实现。强制子类必须实现某些方法，否则创建对象时直接报错。**

```python
from abc import ABC, abstractmethod

class Shape(ABC):
    """抽象类：不能直接创建 Shape 对象"""

    @abstractmethod
    def area(self):
        """计算面积——子类必须实现"""
        pass

    @abstractmethod
    def perimeter(self):
        """计算周长——子类必须实现"""
        pass

    def describe(self):
        """普通方法，子类可以不重写"""
        print(f"面积: {self.area():.2f}  周长: {self.perimeter():.2f}")

# shape = Shape()   # TypeError: 不能实例化抽象类

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14159 * self.radius ** 2

    def perimeter(self):
        return 2 * 3.14159 * self.radius

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

    def perimeter(self):
        return 2 * (self.width + self.height)

# ===== 使用 =====
c = Circle(5)
r = Rectangle(4, 6)
c.describe()   # 面积: 78.54  周长: 31.42
r.describe()   # 面积: 24.00  周长: 20.00
```

### 3.9 组合 vs 继承

**继承是 "is-a"（是一个），组合是 "has-a"（有一个）。**

| 关系 | 关键词 | 例子 | 什么时候用 |
|------|--------|------|-----------|
| 继承 | is-a | Dog is Animal（狗是动物） | 类型关系明确、需要复用接口 |
| 组合 | has-a | Car has Engine（车有发动机） | 组件式关系、更灵活、不暴露父类接口 |

```python
# ===== 组合：Car "有一个" Engine =====
class Engine:
    def __init__(self, horsepower):
        self.horsepower = horsepower

    def start(self):
        print(f"引擎启动! {self.horsepower}马力")

    def stop(self):
        print("引擎熄火")

class Car:
    def __init__(self, brand, engine):
        self.brand = brand
        self.engine = engine    # 组合：Car 包含一个 Engine 对象

    def drive(self):
        self.engine.start()
        print(f"{self.brand} 汽车上路了!")

# ===== 继承：Car 也可以"是"一个 Vehicle =====
class Vehicle:
    def __init__(self, max_speed):
        self.max_speed = max_speed

    def info(self):
        print(f"最高时速: {self.max_speed} km/h")

class ElectricCar(Vehicle, Car):   # 这里只是演示，实际别这样写
    pass

# 真正的推荐写法：同时用继承和组合
class GasCar(Vehicle):      # GasCar "是一个" Vehicle（继承）
    def __init__(self, brand, max_speed, horsepower):
        super().__init__(max_speed)
        self.brand = brand
        self.engine = Engine(horsepower)    # GasCar "有一个" Engine（组合）

    def drive(self):
        self.engine.start()
        print(f"{self.brand} 上路了，最高时速 {self.max_speed} km/h")

car = GasCar("特斯拉", 200, 300)
car.info()     # 来自 Vehicle：最高时速: 200 km/h
car.drive()    # 引擎启动! 300马力  特斯拉 上路了，最高时速 200 km/h
```

### 3.10 完整实战项目：图书管理系统

```python
class Book:
    """书籍类"""
    def __init__(self, book_id, title, author):
        self.book_id = book_id      # 图书编号
        self.title = title          # 书名
        self.author = author        # 作者
        self.is_borrowed = False    # 是否被借出

    def __str__(self):
        status = "已借出" if self.is_borrowed else "在馆"
        return f"[{self.book_id}]《{self.title}》- {self.author} ({status})"


class Reader:
    """读者类"""
    def __init__(self, name, reader_id):
        self.name = name
        self.reader_id = reader_id
        self.borrowed_books = []    # 记录借了哪些书

    def borrow(self, book):
        self.borrowed_books.append(book)

    def return_book(self, book):
        if book in self.borrowed_books:
            self.borrowed_books.remove(book)
        else:
            print(f"错误: {self.name} 没有借过这本书")

    def __str__(self):
        return f"读者 {self.name}({self.reader_id})"


class Library:
    """图书馆——核心管理类"""
    def __init__(self, name):
        self.name = name
        self.books = []         # 馆藏图书列表
        self.readers = {}       # 读者字典（key=读者ID, value=Reader对象）

    def add_book(self, book):
        """上架新书"""
        self.books.append(book)
        print(f"已上架: {book.title}")

    def register_reader(self, reader):
        """注册读者"""
        self.readers[reader.reader_id] = reader
        print(f"已注册: {reader.name}")

    def borrow_book(self, reader_id, book_id):
        """借书逻辑"""
        # 检查读者是否存在
        reader = self.readers.get(reader_id)
        if reader is None:
            print("错误: 读者不存在")
            return

        # 检查书是否存在
        book = None
        for b in self.books:
            if b.book_id == book_id:
                book = b
                break
        if book is None:
            print("错误: 图书不存在")
            return

        # 检查是否已被借出
        if book.is_borrowed:
            print(f"错误: 《{book.title}》已被借出")
            return

        # 执行借书
        book.is_borrowed = True
        reader.borrow(book)
        print(f"借出成功: {reader.name} 借了《{book.title}》")

    def return_book(self, reader_id, book_id):
        """还书逻辑"""
        reader = self.readers.get(reader_id)
        if reader is None:
            print("错误: 读者不存在")
            return

        book = None
        for b in self.books:
            if b.book_id == book_id:
                book = b
                break
        if book is None:
            print("错误: 图书不存在")
            return

        if not book.is_borrowed:
            print(f"错误: 《{book.title}》没有被借出")
            return

        book.is_borrowed = False
        reader.return_book(book)
        print(f"还书成功: {reader.name} 还了《{book.title}》")

    def list_books(self):
        """列出所有图书"""
        print(f"\n===== {self.name} 馆藏 =====")
        if not self.books:
            print("（暂无藏书）")
        for book in self.books:
            print(f"  {book}")

    def list_reader_books(self, reader_id):
        """列出某读者的借阅记录"""
        reader = self.readers.get(reader_id)
        if reader is None:
            print("错误: 读者不存在")
            return
        print(f"\n{reader.name} 当前借阅:")
        if not reader.borrowed_books:
            print("  （无）")
        for book in reader.borrowed_books:
            print(f"  《{book.title}》")


# ===== 运行演示 =====
if __name__ == "__main__":
    lib = Library("星辰图书馆")

    # 上架图书
    lib.add_book(Book("001", "三体", "刘慈欣"))
    lib.add_book(Book("002", "活着", "余华"))
    lib.add_book(Book("003", "小王子", "圣埃克苏佩里"))

    # 注册读者
    lib.register_reader(Reader("张三", "R001"))
    lib.register_reader(Reader("李四", "R002"))

    # 借书 & 还书
    lib.borrow_book("R001", "001")       # 张三借《三体》
    lib.borrow_book("R002", "001")       # 李四借《三体》——已被借出，报错
    lib.borrow_book("R002", "002")       # 李四借《活着》
    lib.list_books()                     # 查看馆藏状态
    lib.list_reader_books("R001")       # 张三的借阅记录
    lib.return_book("R001", "001")      # 张三还《三体》
    lib.borrow_book("R002", "001")      # 李四现在可以借了
    lib.list_books()                     # 再看馆藏状态
```

## 4. 易错点速查表

| 易错点 | 错误示例 | 正确写法 |
|--------|----------|----------|
| 忘记 self | `def bark():` | `def bark(self):` |
| 类属性当实例属性用 | `self.species = "猫"` 写在 `__init__` 外面 | 分清楚是放类体还是 `__init__` |
| self 忘写 | `name = "旺财"`（在 `__init__` 里） | `self.name = "旺财"` |
| 多继承顺序 | `class C(B, A)` 覆盖了不该覆盖的 | 用 `类名.mro()` 检查顺序 |
| 混用 `_x` 和 `__x` | `__name = "张三"` 用于一般私有 | 一般用 `_name`，需要防子类冲突用 `__name` |
| super() 忘加括号 | `super.__init__()` | `super().__init__()` |
| 误解 `__str__` | 没用 `print()` 直接打对象看不到 | `print(obj)` 才会调用 `__str__` |
| 在实例上改类属性 | 误以为改了类属性，其实新建了实例属性 | 改类属性用 `类名.属性 = xxx` |
| 抽象类未实现方法 | `class Square(Shape):` 没有 area() | 所有 `@abstractmethod` 都必须实现 |
| 裸 `except:` | `except: print("出错")` | `except ValueError as e:` 精确捕获 |

### 项目 1：学生成绩管理系统

**要求**：创建 `Student` 类，记录姓名、学号和成绩列表，能添加成绩、算平均分、判断是否及格。

```python
class Student:
    """学生类"""
    def __init__(self, name, student_id):
        self.name = name
        self.student_id = student_id
        self.scores = []           # 成绩列表

    def add_score(self, score):
        """添加一门成绩（0-100之间）"""
        if 0 <= score <= 100:
            self.scores.append(score)
            print(f"{self.name} 添加成绩: {score}")
        else:
            print(f"无效成绩: {score}，必须在 0-100 之间")

    def average(self):
        """计算平均分"""
        if not self.scores:
            return 0
        return sum(self.scores) / len(self.scores)

    def is_pass(self):
        """是否全部及格（每门 >= 60）"""
        return len(self.scores) > 0 and all(s >= 60 for s in self.scores)

    def __str__(self):
        avg = self.average()
        status = "全部及格" if self.is_pass() else "有待提高"
        return (f"{self.name}({self.student_id}) "
                f"成绩: {self.scores}  平均: {avg:.1f}  {status}")
```

```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.__balance = balance

    def deposit(self, amount):
        if amount <= 0:
            print("存款金额必须大于 0")
            return
        self.__balance += amount
        print(f"{self.owner} 存入 {amount}，余额 {self.__balance}")

    def withdraw(self, amount):
        if amount <= 0:
            print("取款金额必须大于 0")
            return False
        if amount > self.__balance:
            print(f"{self.owner} 余额不足! 余额 {self.__balance}")
            return False
        self.__balance -= amount
        print(f"{self.owner} 取出 {amount}，余额 {self.__balance}")
        return True

    def transfer_to(self, other_account, amount):
        """转账：从自己转到另一个账户"""
        print(f"转账 {amount} 元: {self.owner} -> {other_account.owner}")
        if self.withdraw(amount):
            other_account.deposit(amount)
            print("转账成功!")
        else:
            print("转账失败!")

    def __str__(self):
        return f"{self.owner}: 余额 {self.__balance} 元"
```

### 项目 3：形状继承体系

**要求**：父类 `Shape` 定义抽象接口，子类 `Circle`、`Rectangle`、`Triangle` 实现面积和周长计算。

```python
from abc import ABC, abstractmethod
import math

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

    @abstractmethod
    def perimeter(self):
        pass

    def __str__(self):
        return f"{self.__class__.__name__}: 面积={self.area():.2f}, 周长={self.perimeter():.2f}"

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    def area(self):
        return math.pi * self.radius ** 2
    def perimeter(self):
        return 2 * math.pi * self.radius

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    def area(self):
        return self.width * self.height
    def perimeter(self):
        return 2 * (self.width + self.height)

class Triangle(Shape):
    def __init__(self, a, b, c):
        self.a, self.b, self.c = a, b, c
        if not (a + b > c and a + c > b and b + c > a):
            raise ValueError("不能构成三角形!")
    def area(self):
        s = (self.a + self.b + self.c) / 2
        return math.sqrt(s * (s - self.a) * (s - self.b) * (s - self.c))
    def perimeter(self):
        return self.a + self.b + self.c

# 测试：多态——统一处理所有形状
shapes = [Circle(5), Rectangle(4, 6), Triangle(3, 4, 5)]
for s in shapes:
    print(s)
# 输出:
# Circle: 面积=78.54, 周长=31.42
# Rectangle: 面积=24.00, 周长=20.00
# Triangle: 面积=6.00, 周长=12.00
```

### 项目 4：自定义字符串包装类

**要求**：实现 `__str__`、`__len__`、`__eq__`、`__add__`、`__getitem__`，让自定义类用起来像字符串。

```python
class Text:
    def __init__(self, content):
        self.content = content

    def __str__(self):
        return self.content

    def __len__(self):
        return len(self.content)

    def __eq__(self, other):
        if isinstance(other, Text):
            return self.content == other.content
        return False

    def __add__(self, other):
        if isinstance(other, Text):
            return Text(self.content + other.content)
        return Text(self.content + str(other))

    def __getitem__(self, index):
        return self.content[index]

    def to_upper(self):
        return Text(self.content.upper())

# 测试
t1 = Text("Hello")
t2 = Text("World")

print(t1)                # Hello
print(len(t1))           # 5
print(t1 == t2)          # False
print(t1 == Text("Hello"))   # True
print(t1 + " " + t2)     # Hello World
print(t1[1])             # e
print(t1.to_upper())     # HELLO
```
